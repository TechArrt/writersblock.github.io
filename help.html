How to create a post
  1. Press the create button
  2. On a dialog box select either Poem or Devotion
  3. Follow the intructions in the editor

How to upload to post

  1. If in the editor press the POST button in the appbar, else
  2. Press account > select My Devotions or Poems
  3. Select appropriate item
  4. Press POST

How to delete from MyBlock
Warning, as for now once posted, post can not be deleted

  1. If in the editor press the menu icon in the appbar and then delete, else
  2. Press account > select My Devotions or Poems
  3. Select appropriate item
  4. Press delete

How to update post

  1. If in the editor press the POST button in the appbar and follow intructions that follow else
  2. Press account > select My Devotions or Poems
  3. Select appropriate item
  4. Repeat 1
